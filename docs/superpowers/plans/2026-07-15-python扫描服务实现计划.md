# Python扫描服务实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 Windows 64 位客户机上实现一个仅支持 TWAIN 的 Python 本机扫描服务，通过 `http://127.0.0.1:17653` 向 Vue 页面提供设备、参数、任务、页面、缩略图和事件接口。

**Architecture:** 登录用户级托盘主进程运行 FastAPI，并作为 SQLite 唯一写入者；长期存活的 TWAIN 工作子进程独占 `TWAINDSM.DLL`、Data Source 和消息循环。进程间只传递结构化命令、事件和文件路径，驱动使用 `TWSX_FILE + TWFF_JFIF` 直接输出每面一个 JPEG，Pillow 只读取原图生成缩略图。

**Tech Stack:** Python 3.12.13 64 位、FastAPI、Uvicorn、pytwain 2.3.0、Pillow、SQLite、Pydantic v2、pytest、pytest-asyncio、httpx、PyInstaller、Inno Setup。
