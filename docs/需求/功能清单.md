# 功能清单

## 1. 服务运行

- 登录 Windows 后自动启动托盘主进程。
- 固定监听 `127.0.0.1:17653`。
- 提供健康检查、版本和运行信息。
- 托盘显示正常、扫描中、无响应、异常和未启动状态。
- 托盘支持打开数据目录、打开日志目录、重启扫描组件、重启本机服务和退出。

## 2. TWAIN设备

- 加载64位 `TWAINDSM.DLL`。
- 枚举当前用户会话中的全部64位 TWAIN Data Source。
- 返回厂商、产品系列、产品名称和协议版本。
- 保存上次成功使用设备作为默认值。
- 同一时间只允许一个设备执行扫描。
- 目标驱动为 `KODAK Scanner: i2000`（`kds_i2000.inf`）；当前真机为 KODAK i2400，具体可用能力以该驱动实时返回为准。

## 3. Capability

- 查询 `CAP_SUPPORTEDCAPS`。
- 查询 `MSG_QUERYSUPPORT` 和 `TWQC_*`操作位。
- 查询当前值、默认值和允许值。
- 解析 `TW_ONEVALUE`、`TW_ENUMERATION`、`TW_RANGE`、`TW_ARRAY`。
- 返回全部标准和厂商私有 Capability。
- 标准 Capability提供中文映射。
- 私有 Capability没有映射时保留原始编号和值。
- 前端使用固定业务配置字段，不根据全部 Capability 动态生成配置字段。
- Capability 查询结果为固定配置字段提供设备支持状态、候选值、范围、当前值、默认值和查询错误。
- 设置前校验 Capability 存在、Item 类型、枚举集合或范围边界；`TWQC_SET` 操作位保留为诊断信息，不作为固定配置设置的唯一拒绝条件。
- `MSG_SET` 返回状态作为设置结果；设备支持 `MSG_GETCURRENT` 时回读实际值，不支持时记录无法回读，不因此单独判定设置失败。
- 不支持通用 `TWQC_RESET` 时，不调用重置消息，恢复默认值使用固定配置或明确的默认值请求。
- `DAT_CUSTOMDSDATA`只支持整块保存和恢复。

## 4. 扫描方式

- 完全无厂商界面扫描。
- 根据设备能力支持平板、ADF单面、ADF双面。
- ADF无纸正常结束本轮扫描。
- 卡纸、设备断开和驱动异常标记失败。
- 支持在已停止、失败或完成任务中继续补扫。
- 双面扫描时正面和背面分别生成独立页面。

## 5. 图像传输

- 仅支持 `TWSX_FILE`。
- 固定 JPEG/JFIF单页输出。
- 每一面生成独立 `pageId`和 JPEG文件。
- 不降级到 `TWSX_MEMORY`或`TWSX_NATIVE`。
- 不支持 `TWSX_FILE + JPEG` 的设备直接返回不兼容。
- JPEG质量通过 `ICAP_JPEGQUALITY`动态查询和设置。

## 6. 缩略图

- 原图由 TWAIN驱动直接写入，Python不重新编码原图。
- Python使用 Pillow额外生成 JPEG缩略图。
- 缩略图默认最长边320像素、质量75。
- 缩略图生成失败时保留原图并记录预览异常。
- 缩略图可以重新生成，原图摘要必须保持不变。

## 7. 扫描任务

- 支持保存多个历史任务。
- 同一时间只允许一个任务扫描。
- 状态包括 `CREATED`、`SCANNING`、`STOPPING`、`STOPPED`、`COMPLETED`、`FAILED`、`CANCELLED`。
- SQLite保存任务、页面、设备和 Capability快照。
- 服务重启后恢复历史任务。
- 扫描中任务重启后转为停止或失败，不自动继续扫描。
- 失败、取消和中断任务不自动删除。

## 8. 页面文件

- 查询任务页面。
- 增量查询新增页面。
- 读取页面详情。
- 读取缩略图。
- 读取原始 JPEG。
- 删除已完成页面。
- 删除页面时同步处理原图、缩略图和 SQLite记录。
- 删除页面后其他页面的 `pageId`和原始 `sequence`不变。

## 9. 任务删除

- 明确调用删除接口后才删除任务。
- 删除任务记录、页面记录、原图、缩略图和任务目录。
- 删除失败不返回虚假成功。
- 不设置按时间自动清理。

## 10. 事件

- 服务启动。
- 工作进程启动和重建。
- 设备列表变化。
- 任务创建、开始、停止、完成和失败。
- 页面开始、完成和删除。
- 任务删除。
- 第一版不承诺单页扫描百分比。

## 11. 明确排除

- 业务系统登录和权限。
- 档案查询和分组。
- 业务上传批次。
- 正式原文挂接。
- 原图旋转、裁剪、纠偏、重新压缩、OCR和PDF处理。
- 其他扫描协议和操作系统。
